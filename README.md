## ✨ Key Features

- **Interactive 3D Elements**: Powered by Three.js and Spline.
- **Fluid Animations**: Smooth scrolling with Lenis and scroll-triggered animations via GSAP.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Content**: Sections for Career, Education, Work, and more.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/satyarajreddy/portfolio_website.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
